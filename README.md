## 🚀 Featured Project

### 🔧 [DevOps Pipeline — Uptime Kuma](https://github.com/jimmyj13/uptime-kuma)

> Production-style DevOps pipeline built around an open-source Node.js monitoring tool

- 🐳 **Docker** — Multi-stage Dockerfile, Docker Compose for local dev
- ⚙️ **CI/CD** — GitHub Actions pipeline: build → push to Docker Hub on every commit (tagged with `latest` + commit SHA)
- ☸️ **Kubernetes** — Minikube deployment with liveness/readiness probes, PVCs, ConfigMaps, Nginx Ingress
- 📊 **Observability** — Prometheus scraping every 15s, Grafana dashboards (heap memory, event loop lag, connections), Alertmanager Slack notifications
- 🔒 **Security** — Kubernetes Secrets for credential management
